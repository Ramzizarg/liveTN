## Setup

- **Code or link**: Use the key icon in the header to enter your IPTV code or M3U playlist link (like IPTV Smarters / XCIPTV).
- **Reload channels**: After saving a link, use "Reload channels" or reload the app to load channels.
- **EPG**: If your provider supports it, the TV guide (Now / Next) appears on the watch page.
